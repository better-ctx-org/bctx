## [0.1.7]

### Added
- **Wave 2 skills** (21 new): Parallax, Blueprint, Pinpoint, Crossroads, Drift, Panorama, Ripple, Meridian, Forecast, Unfold, Thermal, Echo, Ledger, Steward, Dispatch, Arbiter, Diviner, Crucible, Scanner, RelayCtx, Witness.
- **Wave 3 skills** (5 new): Flux, Harvest, Pathfinder, Render, Scribe.
- Total skill count: **41**.

### Changed
- Improved tree-sitter parsing performance across all 17 supported languages.

## [0.1.0]

### Added
- Initial public release.
- **Wave 1 skills** (15): Sieve, Cartograph, Chisel, Sediment, Prism, Compass, Condenser, Archivist, Scout, Chronicler, Alchemist, Sentinel, Cartridge, Resonator, Surveyor.
- MCP stdio + HTTP/SSE transports.
- 24 native agent integrations via `bctx init --agent <name>`.
- Install scripts for macOS, Linux, and Windows.
- Homebrew tap and npm package (`bctx-bin`).
